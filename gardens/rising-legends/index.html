Rising Legends [Garden]

[ project ] [ rising-legends ] [ garden ]

This page will serve as the current state of Rising Legends, a Persistent Browser-Based Game that I’m working on.

Features

  • Skills - All users get the “core” skills: bladed, blunt, one handed mastery, two handed mastery, destruction magic, restoration magic, light armour mastery, heavy armour mastery. These are all passive and improve based on use.

News

2023-05-07

Started the rework on inventory to support updating skills dynamically. All skills are “affected by” a skill. For example, a Short Sword is affected by your One Handed Mastery skill and your Bladed Mastery skill.

2023-05-06

Finally set up a page where I can track the current status of the game (known bugs, changelog, etc.)
